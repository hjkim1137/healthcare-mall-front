# 🛒 LifeCare Mall – **풀스택 이커머스 플랫폼**

React · Spring Boot · MySQL · AWS 기반의 **2인 팀 프로젝트**로,  
검색·장바구니·바로구매·주문·인증·이미지 업로드까지  
**실제 서비스 수준의 기능**을 구현한 풀스택 이커머스 플랫폼입니다.

---

## 👥 프로젝트 개요
- **2인 협업 풀스택 개발**
- FE/BE 병렬 개발, 공통 API 설계, Git-flow 적용
- 실 서비스 운영을 고려한 아키텍처 및 UX 설계

---

## 📌 기획 배경
- 실무형 기술 스택으로 **서비스 전 과정 경험**
- 건강 식품·의료기기 기반 이커머스 설계
- 인증·이미지 관리·주문 흐름 등 실서비스 수준 기능 구현
- 현재 **관리자(Admin) 페이지 고도화 진행 중**

---

## 🛠 기술 스택

### **Frontend**
- React, TypeScript

### **Backend**
- Spring Boot  
- Spring Security  
- JPA (Hibernate)

### **Database**
- MySQL

### **Infra**
- AWS S3 (상품 이미지 저장)

---

## 🚀 주요 기능

### 🔹 유저(User)
- 회원가입 / 로그인 / 로그아웃  
- **메일 기반 비밀번호 찾기(임시 PW + 토큰 만료 처리)**  
- 마이페이지 & 주문 내역 조회

### 🔹 상품(Product)
- 상품 검색 / 카테고리별 조회  
- **장바구니 실시간 금액 반영**  
- **바로구매** (sessionStorage 기반 상태 유지)

### 🔹 주문(Order)
- 수량 변경 시 **상품별/총 금액 자동 계산**
- 전체 선택 / 선택 삭제
- 로그인 정보 기반 **주문자 자동 입력**
- **‘주문자와 동일’ 버튼으로 배송지 자동 채움**

### 🔹 관리자(Admin) *(개발 중)*
- 상품/주문/회원 관리
- 데이터 검증 및 관리자 권한 처리

---

## 🎯 프로젝트 특징 (핵심 강점)

1. **사용자 경험(UX) 강화**  
   - 자동 입력, 클릭 최소화, 실시간 반응형 UI

2. **보안·무결성 강화**  
   - bcrypt 비밀번호 해싱  
   - 인증 기반 주문·장바구니 보호  
   - 토큰 기반 비밀번호 재설정

3. **실 서비스 구조 반영**  
   - AWS S3 이미지 업로드  
   - 검색 → 장바구니/바로구매 → 주문 → 결제 흐름 구현

4. **협업 중심 개발**  
   - API 문서화  
   - Git-flow 전략  
   - 코드 스타일/구조 통일
